## Getting Started

To get a local copy up and running, fallow these simple steps:

### Prerequisites

- Ruby 3.0.1 or newer
- Rails 7.0.4 or newer
- Git
- NPM
- PostgresSQL
- Web browser

### Setup

- git clone git@github.com:KaskMIL/hello-rails-react.git
- cd hello-rails-react

### Install

- bundle install
- npm install

### Usage

Execute command

bin/dev
Then browse https://localhost:3000
